# ViaOS
An x86 Operating System built on the VD Kernel (VDK) by Via.

# Complile
> cd ViaOS

``` make ```

# Run
Copy mxos.bin kernel from build/ to your working directory and:

``` qemu-system-i386 -kernel mxos.bin ```
